این پروژه یک **دستیار هوشمند فروش پوشاک** است که با استفاده از **پایتون، Gradio و OpenAI Agents** پیاده‌سازی شده است.
کاربر می‌تواند هم **سؤالات عادی** بپرسد و هم **فرآیند ثبت سفارش چندمرحله‌ای** را در یک چت واحد انجام دهد.

### قابلیت‌های اصلی

*پشتیبانی مکالمه‌ای (سلام، سوالات متداول، اطلاعات عمومی)
*جستجوی محصولات از داده‌های محلی (JSON + جستجوی برداری)
***ماشین حالت قفل‌شده برای سفارش** (مرحله‌به‌مرحله و قابل‌کنترل)
*استفاده از OpenAI Agent به‌همراه ابزارها
*تفکیک شفاف بین:
  * پاسخ‌های تولیدی مدل زبانی
  * منطق قطعی کسب‌وکار (سفارش)
*پاسخ‌دهی استریمی با افکت تایپ
*تشخیص خودکار نیت کاربر (سفارش یا سوال عادی)
*ذخیره سفارش‌ها در فایل `orders.json`

### نکات معماری مهم

* معماری **تک‌عامل (Single Agent)** با ابزارها
* منطق سفارش کاملاً خارج از LLM پیاده شده تا از پاسخ‌های ساختگی جلوگیری شود
* رابط چت Gradio
* طراحی Async با استفاده از `Runner.run()` 

این ساختار باعث می‌شود سیستم **پایدار، قابل توسعه و مناسب استفاده واقعی در پشتیبانی مشتریان** باشد.


### تکنولوژی های استفاده شده

* **تست**: Python 3
* **LLM**: OpenAI
* **رابط کاربری**: Gradio 6.2.0


### مراحل راه اندازی و اجرا:

1. **مخزن را کلون کنید**
   ```bash
   git clone https://github.com/MaryamMirbagheri/LLM_engineering_agentic_ai.git
   cd .\community_contributions\maryam_mirbagheri\final_project\

2. **ساخت محیط مجازی (virtual environment)**
* برای نصب و جداسازی وابستگی های پروژه یک محیط مجازی بسازید
  ```bash
  py -m venv venv

3. **فعال سازی محیط مجازی**
* محیط مجازی را بسته به سیستم عامل خود فعال کنید
  ```bsh
  Windows:
  venv\Scripts\activate

  Linux+ macOS:
  source venv/bin/activate

4. **نصب تمام کتابخانه ها**
* برای نصب تمام کتابخانه های مورد نیاز به همراه ورژن های مطابق با این پروژه با دستور زیر نصب کنید:
  ```bash
  pip install -r requirements.txt

* بعد از نصب تمامی کتابخانه ها محیط مجازی را غیرفعال کنید.
  ```bash
  deactivate

5. **API کلید های**
* کلید های اختصاصی خود را در فایل .env به صورت متغیر محلی ذخیره کنید و کلید های اختصاصی خود را با توجه به فایل .env.example ذخیره کنید.

6. **اجرای برنامه**
* برای اجرای برنامه و رابط گرافیکی دستور زیر را در ترمینال وارد کنید.
